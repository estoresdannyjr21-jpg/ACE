Company logo — where to save
=============================

To show your company logo on the LOGIN page and in the TOP BAR:

  Exact location:   web/public/logo.png
  Full path example:  [project]/ace-truckers-erp/web/public/logo.png

  File name:  logo.png  (or logo.jpg if you prefer; then change LOGO_PATH in App.tsx to /logo.jpg)

Recommended:
  - PNG or JPG
  - Login page: about 200–400px wide (height auto), so it looks clear above the form
  - Top bar uses the same image and scales it down (max height 36px)

If the file is missing, the app shows "Ace Truckers Corp" as text instead.
